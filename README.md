### Logitech Gaming Controller F310 - Button Layout:

**Axes:**  
0 = left stick X  
1 = left stick Y (inverted)  
2 = right stick X  
3 = right stick Y (inverted)  
 
**Buttons:**  
1 = X, 2 = A, 3 = B, 4 = Y  
5 = Left Bumper (LB)  
6 = Right Bumper (RB)  
7 = Left Trigger (LT)  
8 = Right Trigger (RT)  
9 = Back Button  
10 = Start Button  
11 = Left Stick Press (Click In)  
12 = Right Stick Press (Click In)  

**POV:**  
0 = Up  
90 = Right  
180 = Down  
270 = Left
